# ECS-408/608 2026 Operating System Assignments

This repository contains operating system assignment work completed for IISER Bhopal.

## Repository structure

- `assignment 1/`
  - CPU simulation and printing code
  - Disk read/write programs
  - Memory simulation programs
  - Supporting files and traces
  - `disk-files/` contains sample disk files used by the assignments

- `assignment 2/`
  - `q1`, `q2`, `q3`, `q4` folders with corresponding C programs and outputs
  - Assignment 2 source code and result files

- `assignment 3/`
  - Producer/consumer implementations for multiple questions
  - `ques1/`, `ques2/`, `ques3/` directories with code and binaries

- `assignment 4/`
  - Multiple OS lab programs
  - `ques1/`, `ques2/`, `ques3/` directories with source code and compiled outputs

## Notes

- The repository is intended to store OS assignment code and related artifacts.
- Use the directories above to locate each assignment's source files and outputs.
